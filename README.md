# Weather App

A responsive weather application that provides real-time weather information for different locations using live weather APIs.  
The application displays essential weather details such as temperature, humidity, weather conditions, and wind speed through a clean and interactive interface.

---

## Features

- Search Weather by City
- Real-time Weather Updates
- Temperature & Weather Condition Display
- Humidity and Wind Speed Information
- Dynamic Weather Data Fetching using API
- Responsive User Interface
- Error Handling for Invalid Locations
